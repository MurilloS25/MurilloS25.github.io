Para definir una funcion en "JS" hacemos uso de la palabra reservada function, seguido de esto escribiremos el nombre de la funcion y se ponen los paréntesis con los parámetros para finalizar abriendo el cuerpo con el uso de los corchetes.
Ejemplo:
function NombreDeFuncion(n Parámetros){
Instrucciones de la función.
}

Para llamar una funcion de javaScript desde nuestro HTML debemos hacer uso de alguno de los eventos html y dentro de este llamar la función.
Ejemplo:
< Etiqueta onclick="NombreFuncion()"> Texto de la Etiqueta < Etiqueta Cierre>

En este caso se puso de ejemplo el evente onclick, pero se puede usar cualquier otro de los disponibles

Asignación y Comparacion

En "JS" el simbolo de "=" tiene tres maneras de ser utilizado.

La primera es para asignar un valor a una variable. (Solo se usa un signo)

Ejemplo:
var hilera = "Hola"

Las otras dos son para comparar.

Se utilizan dos simbolos de "=" para comparar dos variables ignorando el tipo de dato

Ejemplo:
1 == "1"
Esto dara como resultado "true" porque son lo mismo y no se valida el tipo

Se utilizan tres simbolos de "=" para comparar dos variables pero tomando en cuenta si son del mismo tipo

Ejemplo:
1 === "1"
Esto dara como resultado "false" porque son de distintos tipos

Object Number

Number es un objeto primitivo envolvente que permite representar y manipular valores numéricos. Los principales usos del objeto Number(valor) son convertir un string u otro valor a uno de tipo numérico; si el argumento no puede ser convertido a un número, devuelve NaN.

Object Math

Math es un objeto incorporado que tiene propiedades y métodos para constantes y funciones matemáticas. No es un objeto de función. A diferencia de los demás objetos globales, el objeto Math no se puede editar. Todas las propiedades y métodos de Math son estáticos.

Object Date

Como su nombre lo indica es un objeto que añade diversas funcionalidades muy útiles respecto a las fechas.

Cadenas Literales

Las plantillas literales son cadenas literales que habilitan el uso de expresiones incrustadas. Con ellas, es posible utilizar cadenas de caracteres de más de una línea, y funcionalidades de interpolación de cadenas de caracteres. Se caracterizan por el uso de ` ` estas comillas.
Ejemplo: `texto de cadena de caracteres ${variable} texto adicional`.

Arrays

El objeto Array de JavaScript es un objeto global que es usado en la construcción de arrays, que son objetos tipo lista de alto nivel. Los arrays son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación. Tanto la longitud como el tipo de los elementos de un array son variables.

Los arrays se definen de las siguientes manera:
let array = [1,2,3,4,5]

Arrays Tipados

Los arreglos tipados en JavaScript son objetos similares a arreglos que proporcionan un mecanismo para leer y escribir datos binarios sin procesar en búferes de memoria.
